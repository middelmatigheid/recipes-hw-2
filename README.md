# Recipes - homework 2

# About the project

It's just a homework about Git, OOP and testing in python. See .ipynb file for more details

You can see developing history and different versions via commits

See tests for classes Ingredient, Recipe and ShoppingList via pytest in file test-recipes.py. Because you can't
import from .ipynb to .py, code from .ipynb was copied in .py

# Downloading the project

To download the project open terminal and use commands

```bash
git clone https://github.com/middelmatigheid/recipes-hw-2.git
cd recipes-hw-2
```

To rune the tests use commands

```bash
pip install -r requirements.txt
pytest
```

# Project structure

